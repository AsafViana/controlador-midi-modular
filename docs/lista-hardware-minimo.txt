🎛️ *HARDWARE MÍNIMO - Controlador MIDI Modular*

Pra testar todas as funcionalidades do sistema, incluindo comunicação entre módulos.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📦 *MÓDULO PRINCIPAL (obrigatório)*

• 1x ESP32-S3-DevKitC-1 (N16R8) — cérebro do sistema
• 1x Display OLED SSD1306 128x64 I2C — tela de interface
• 3x Botão táctil (push button) — navegação UP/DOWN/SELECT
• 4x Potenciômetro 10kΩ linear — controles MIDI (Volume, Pan, Modulação, Extra)
• 1x LDR + 1x resistor 10kΩ — sensor de luz
• 1x Cabo USB-C — alimentação + USB MIDI

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎵 *MIDI DIN (opcional)*

Se quiser testar saída MIDI por cabo DIN (5 pinos):

• 1-2x Conector DIN 5 pinos (fêmea)
• 1x Optoacoplador 6N138
• 2x Resistores (220Ω e 470Ω)
• 1x Diodo 1N4148

⚠️ O MIDI USB já funciona só com o cabo USB-C, então pode pular isso no início.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔗 *MÓDULO REMOTO I2C (pra testar comunicação entre módulos)*

• 1x Arduino Nano ou ATtiny85 — módulo escravo I2C
• 1-2x Potenciômetros no módulo remoto
• 2x Resistores 4.7kΩ — pull-ups pro barramento I2C

O módulo escravo responde nos endereços 0x20 a 0x27 com dois comandos:
  → 0x01 = envia descritor (nome e tipo dos controles)
  → 0x02 = envia valores atuais (0-127)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 *PINAGEM DO ESP32-S3*

GPIO 1 → Pot Volume
GPIO 2 → Pot Pan
GPIO 3 → Pot Modulação
GPIO 5 → I2C SDA (display + módulos)
GPIO 4 → I2C SCL (display + módulos)
GPIO 6 → Sensor Luz (LDR)
GPIO 7 → Pot Extra
GPIO 9 → MIDI TX (DIN OUT)
GPIO 10 → MIDI RX (DIN IN)
GPIO 11 → Botão UP
GPIO 12 → Botão DOWN
GPIO 13 → Botão SELECT
GPIO 0 → LED indicador

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Com isso dá pra testar:
• Leitura de potenciômetros e sensor
• Navegação por menus no display
• Envio de MIDI CC via USB
• Envio de MIDI CC via DIN (se montar o circuito)
• Comunicação I2C com módulo remoto
• Descoberta automática de módulos
• Mapeamento de CC pelo menu
• Factory reset
